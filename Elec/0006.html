晶体管

一、放大模式下的重要公式

$i_C = I_Se^{v_{BE}/V_T}$ $i_C = \beta i_B$ $i_E = (\beta + 1)i_B$ $\beta = \frac{\alpha}{1 - \alpha}$ $\alpha = \frac{\beta}{1+\beta}$

$g_m = \frac{\Delta I_{out}}{\Delta V_{in}}(直流) = \frac{i_{out}}{v_{in}}(交流) = \frac{i_c}{v_{be} }= \frac{I_C}{V_T}(求解直流下的I_C)$

$\pi$ 等效：

$r_\pi = \frac{v_{be}}{i_b} = \frac{V_T}{I_B}(求解直流下的I_B) = \frac{\beta}{g_m}$

$T$ 等效：

$r_e = \frac{v_{be}}{i_e} = \frac{V_T}{I_E}(求解直流下的I_E) = \frac{\alpha}{g_m}$

二、共射放大电路

$A_v = \frac{v_o}{v_i} = \frac{-i_c(R_C//R_L}{i_br_\pi} = \frac{-\beta i_b(R_C//R_L}{i_br_\pi} = \frac{-\beta (R_C//R_L)}{r_\pi}$
$R_i = R_1//R2//r_\pi$
$R_o = R_c$
（因为算等效输出电阻的时候，电压源$v_s$置零，所以受控源也跟着为0）
特点：输入电阻中等，输出电阻大，频带窄！放大电压和电流！

三、共基放大电路

$A_v = \frac{v_o}{v_i} = \frac{-\alpha i_eR_C}{-i_er_e} = \frac{\alpha R_C}{r_e} = g_mR_C$
$A_i = \frac{i_c}{i_e} = \alpha \approx 1$ 电流缓冲器
$R_i = r_e$
$R_o = R_c$ （因为算等效输出电阻的时候，电压源$v_s$置零，所以受控源也跟着为0，电流源为零，看成开路）
特点：输入电阻小，频带宽！只放大电压，不放大电流！

四、共集放大电路

$A_v = \frac{v_o}{v_i} = \frac{i_e(R_L//r_o)}{i_e(r_e + R_L//r_o)} = \frac{R_L//r_o}{r_e + R_L//r_o} \approx 1$ 射极跟随器
$R_i = \frac{v_i}{i_b} = \frac{i_e(r_e + R_L//r_o)}{i_e/(\beta + 1)} = (\beta + 1)(r_e + r_o//R_L)$
$R_o = r_o//(r_e + \frac{R_s}{\beta + 1})$ 走两路到地

特点：输入电阻大，输出电阻小，频带中等！只放大电流，不放大电压！

场效应晶体管

一、放大模式下的重要参数

$g_m = \frac{i_d}{v_{gs} }= \frac{I_D}{(V_{GS}-V_T)/2}$
$\pi$ 等效：

$T$ 等效：

二、共源放大电路

$A_v = \frac{v_o}{v_i} = \frac{-g_mv_{gs}R_D//R_L}{v_{gs}} = -g_mv_{gs}R_D//R_L$
$R_i = R_G$
$R_o = R_D$ 从$R_L$看进去
特点：非常高的输入电阻、适中的电压增益和相当高的输出电阻，频带窄。

三、共漏放大器

$A_v = \frac{v_o}{v_i} = \frac{-g_mv_{gs}R_S//R_L}{v_{gs} + g_mv_{gs}R_S//R_L} = \frac{-g_mR_S//R_L}{1 + g_mR_S//R_L} \approx 1$ 源极跟随器
$R_i = R_G$
$R_o = R_S//\frac{1}{g_m}$ 在算输出电阻的时候，加上外加电源，$v_s$置零，但是$S$点的电位不为零，所以$V_{GS}$不为零。
特点：非常高的输入电阻、相当低的输出电阻和接近于1的电压增益，频带中等。

四、共栅放大器

$A_v = \frac{v_o}{v_i} = \frac{-g_mv_{gs}R_D//R_L}{v_{gs}} = g_mR_D//R_L$ 源极跟随器
$R_i = R_S//\frac{1}{g_m}$
$R_o = R_D$ ????我算出来是$R_S//R_D$
特点：相当低的输入电阻、很大的输出电阻和接近于1的电流增益，且高频特性较好，频带宽。
