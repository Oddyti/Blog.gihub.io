二极管

PN结

一、PN结的形成

  1. 扩散运动（多子）
  2. 空间电荷区：耗尽层、阻挡层、PN结
  3. 漂移运动（少子）
  4. 空间电荷区：阻碍多子扩散、加快少子扩散

    二、PN结的特性：单向导电性

  • 外加正向电压等于增强扩散运动
  • 外加反向电压截止，增大耗尽层（并不是没有电流，此时加强了少子运动（漂移运动），有mA级别的反向饱和电流，对温度很敏感）
  • 二极管做正向导通时候一定要加限流电阻，防止耗尽层烧掉

    三、PN结的电流方程

    $i = I_s(e^{\frac{V}{V_T}}-1)$

$V_T = 25mV$

四、PN结的伏安特性

$i = I_s(e^{\frac{V}{V_T}}-1)$

  • 正向指数关系
  • 反向饱和电流：$I_s$
  • 反向击穿
    • 雪崩击穿
    • 齐纳击穿

二极管的主要参数

  • $I_F$ 最大整流电流
  • $V_R$ 最大反向工作电压，一般是反向击穿电压的一半（余度，保证补击穿）
  • $I_R$ 未击穿时候，反向电流
  • $f_M$ 最高工作频率，因为PN结有电容效应，频率高时，PN近似导通的容抗，单向导电性被破坏。高频电路要注意。

二极管的等效电路

  • 伏安特性折线化

  • (a)是绝对理想的二极管，舍弃掉了导通电压

  • (b)导通电压导通后近似不变

  • (c)

  • 小信号加直流时，直流让二极管导通，从而小信号可以通过，即让小信号在特定的点工作

    二极管的微变等效和稳压二极管

    微变等效

    其中$I_D$为静态电流

    所以同一个二极管在不同的静态工作情况下工作对应的动态电阻不同。

    即二极管微变等效分析：

    1. 寻找直流（静态工作，去掉交流）的$I_D$
    2. 求出$r_d$
    3. 等效为$r_d$，去掉直流，求出$i_d$
    4. 总电流$i_D = i_d + I_D$
