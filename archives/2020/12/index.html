基电collector: 收集载流子，参杂浓度低

放大基本原理

放大基本要求：不失真

放大条件

$R_b$限流电阻，由PN结的伏安曲线，防止电流冲出天际

$V_{BB}$和$V_{CC}$的极性如图，且保证$V_{CC}>V_{BB}$，才使得满足如下条件

==条件：发射结正向偏置，集电结反偏==

$I_C$这个放大电流由$V_{CC}$产生，这样的条件下

内部载流子的运动

发射区正偏

发射区是高参杂浓度，所以发射区的自由电子（N）向基区扩散，基区的空穴向发射区扩散。形成两个电流：

$I_{EN}$、$I_{EP}$ 由于发射区参杂浓度远大于基区，所以$I_{EP} << I_{EN}$

基区

基区薄且参杂浓度低，从发射极而来的这一批自由电子只有小部分在基区和空穴复合，同时VBB会补充，继续和下一批的同样数目的自由电子复合，这样以来$I_{EN}$分成的$I_{BN}$和$I_{CN}$成固定的比例。

集电区

基电结反偏，所以收集自由电子。同时自己会加强自己的漂移运动，产生$I_{CBO}$电流，很小。

放大系数

  1. $\overline{\beta} = \frac{I_{CN}}{I_{BN}} = \frac{I_C - I_{CBO}}{I_B + I_{CBO}} \approx \frac{I_C}{I_B}$

  2. $\beta = \frac{\Delta I_C}{\Delta I_B}$

  3. 当$I_B = 0$时候，依然有$I_C$此时$I_C$称为$I_{CEO}$，穿透电流

  4. $I_{CBO}$

  5. 共基时，$\overline{\alpha} = \frac{I_C}{I_E} = \frac{\overline{\beta}}{1+\overline{\beta}}$

    ==共用哪个极，就是另外两个极的电流之比==

1.3.3 BJT共射放特性曲线

输入特性

即研究$i_b = f(V_{BE})|_{V_{CE} = 常数}$的关系

特点：

$V_{CE}$越小，往左移；$V_{CE}>1$几乎不动。

说明：

$i_B$受$v_{BE}$控制

输出特性

即研究$i_c = f(v_{CE})|_{i_B = 常数}$的关系

当$v_{CE}$大于某个值，集电结反偏，$i_C$不变，和$i_B$成固定比例，放大状态。

饱和区两者不成比例，也即是说$i_B$不受$i_C$控制。

  1. 放大区：发射结正偏，集电结反偏，$i_B$、$i_C$成比例
  2. 截止区：双结反偏，CE断路
  3. 饱和区：双结正偏，即$\beta I_B > I_{Cmax}$

二极管

二极管

PN结

一、PN结的形成

  1. 扩散运动（多子）
  2. 空间电荷区：耗尽层、阻挡层、PN结
  3. 漂移运动（少子）
  4. 空间电荷区：阻碍多子扩散、加快少子扩散

    二、PN结的特性：单向导电性

  • 外加正向电压等于增强扩散运动
  • 外加反向电压截止，增大耗尽层（并不是没有电流，此时加强了少子运动（漂移运动），有mA级别的反向饱和电流，对温度很敏感）
  • 二极管做正向导通时候一定要加限流电阻，防止耗尽层烧掉

    三、PN结的电流方程

    $i = I_s(e^{\frac{V}{V_T}}-1)$

$V_T = 25mV$

四、PN结的伏安特性

$i = I_s(e^{\frac{V}{V_T}}-1)$

  • 正向指数关系
  • 反向饱和电流：$I_s$
  • 反向击穿
    • 雪崩击穿
    • 齐纳击穿

二极管的主要参数

  • $I_F$ 最大整流电流
  • $V_R$ 最大反向工作电压，一般是反向击穿电压的一半（余度，保证补击穿）
  • $I_R$ 未击穿时候，反向电流
  • $f_M$ 最高工作频率，因为PN结有电容效应，频率高时，PN近似导通的容抗，单向导电性被破坏。高频电路要注意。

二极管的等效电路

  • 伏安特性折线化

  • (a)是绝对理想的二极管，舍弃掉了导通电压

  • (b)导通电压导通后近似不变

  • (c)

  • 小信号加直流时，直流让二极管导通，从而小信号可以通过，即让小信号在特定的点工作

    微变等效

    其中$I_D$为静态电流

    所以同一个二极管在不同的静态工作情况下工作对应的动态电阻不同。

    即二极管微变等效分析：

    4. 总电流$i_D = i_d + I_D$
